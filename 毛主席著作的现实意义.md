# 中国革命战争的战略问题
此文用翔实生动的事例给我们讲了深刻的哲学道理。辩证法不是毛主席发明的，但如果不能把它讲得如此具体生动，什么时候才能六亿神州尽尧舜？
我很早就看过马恩原著，两位导师都适用理论上的例子解释辩证法，比如微积分中体现的整体与局部的关系，量变与质变的关系等。
看了毛主席的文章，比如关于战略的持久性和战役的速决性的讨论，才领略到主席教学的魅力。

>对于敌人的一次有计划的“围剿”​，如果我们没有必要的和充分的准备，必然陷入被动地位。临时仓卒应战，胜利的把握是没有的。因此，在和敌人准备“围剿”同时，进行我们的反“围剿”的准备，实有完全的必要。我们队伍中曾经发生过的反对准备的意见是幼稚可笑的。

实际工作中，其实不一定是反对准备，而是没有足够水平去准备，以及也没有体会到准备的必要性。

# 抗日游击战争的战略问题（1938年5月）
虽然这篇文章讨论的是面对强大外敌侵略的战略问题，里面有写观点可以用在很多别的领域，比如科研工作中。虽然科技时刻在进步，但人类对大自然的理解还很有限。如果把大自然比作给我们出“难题”的对手，这个对手无疑是强大的，而我们自己的知识手段往往是有限的。“吾生也有涯，而知也无涯。以有涯随无涯，殆已！”教员的著作里，有很多值得学习的工具，让我们有机会摆脱“殆已”的宿命。具体地说，如果把一般的科研或者研发工作必做针对未知领域的战争，那么游击战就是探索性更强的早期研发工作。

>为什么提起游击战争的战略问题

这是第一章的标题，解释了这篇文章的出发点。放到具体工作中，虽然大多数人都是牛马打工人，但毛主席多次强调，战略眼光和战术水平不是人人都有的，但是可以通过训练获得的。不管是谁，如果不使用正确的方法论，也是不可能得到正确的结论的。正确的方法论的来源就是对更高层次的战略问题的理解。有了对战略问题的理解，就有正确的发挥主观能动性的基础。另外，即使是从事比较专一具体的工作，也许在整个项目这个战略下，只算一个小的战役，但是这个战役本身还是要分解成更多的战斗。在战役和战斗的关系上，战役就又是战略层面的了。所以对战略问题的正确理解，也能帮助我们更有效的完成具体的工作。

>战争的基本原则是　保存自己消灭敌人

这是本篇第二章的标题，它给出了战争，而不单是游击战争的基本原则。以科研工作为例，保存自己就是掌握并保持已知的技能、技术或者流程，消灭敌人就是回答未知的问题、发展并掌握新的技能或者流程。

>为达到这个目的，在军队本身的行动上，就表现为：一方面，尽可能地保存自己的力量；另一方面，尽可能地消灭敌人的力量。何以解释战争中提倡勇敢牺牲呢？每一战争都须支付代价，有时是极大的代价，岂非和“保存自己”相矛盾？其实一点也不矛盾，正确点说，是相反相成的。因为这种牺牲，不但是为了消灭敌人的必要，也是为了保存自己的必要——部分的暂时的“不保存”​（牺牲或支付）​，是为了全体的永久的保存所必需的。

在科研领域，所谓牺牲往往体现在实验的失败以及因此带来的时间或物质上的损失。如果不愿意付出这个代价，而只是重复已知的东西，最后的结果就是个人能力停滞不前，产品开发缓慢，市场开拓停止等等。逆水行舟，不进则退。这种保守主意最终会让个人或团队丧失竞争力，连继续从事重复劳动的机会都没有了。

>所以问题是：应该采取些什么方针或原则才能达到保存或发展自己和消灭敌人的目的呢？总的说来，主要的方针有下列各项：​（一）主动地、灵活地、有计划地执行防御战中的进攻战，持久战中的速决战和内线作战中的外线作战；​（二）和正规战争相配合；​（三）建立根据地；​（四）战略防御和战略进攻；​（五）向运动战发展；​（六）正确的指挥关系。这六项，是全部抗日游击战争的战略纲领，是达到保存和发展自己，消灭和驱逐敌人，配合正规战争，争取最后胜利的必要途径。

这是第三章的绝大部分内容。这些具体的方针，很多都能在科研工作中找到对应。（一）防御战种的进攻战，持久战中的速决战和内线作战中的外线作战：人的知识往往有限而且彼此联系，就像是在一个圈子里，而圈子外面则是更加广袤的未知的需要探索的地方。因此，科研的出发点是保证已知的范围不再缩小或者已经获得的技能不会丢失，这就是防御战。但这个防御战里面必须有进攻的成分，不然就是重复劳动，不会增加我们的知识。又因为圈子外面的知识范围广深度大，这个探索过程必然是丑就的，即使是相对具体的产品研发，需要克服的困难也很多，也应该做打持久战的准备。当时持久战是由一个个速决战组成的，而这个速决战就是短期的任务，这个短期的任务要争取速战速决。如何才能速战速决的，一个原则就是集中优势兵力打歼灭战。在科研活动中，这个优势兵力往往体现在变量控制和操作复杂度。当实验只去研究一个或者很少的变量的效果时，结果就比较容易解释，从而更容易增加我们的知识。当实验操作比较容易时，也就不易犯错误，也更容易增加实验通量。这些条件，从战争角度讲，就是内线作战中的外线作战————虽然我们处在未知的包围下，属于内线作战，但在具体问题上，可以形成局部的反包围，实现相对多的已知面对相对少的位置，也就是局部的外线作战。（二）和正规战争相配合：游击战有点像科研里探索性的活动，它会帮助我们建立较小的分散的独立根据地，而正规战则是更加严格的瀑布式研发过程，遵循项目计划的预算和时间线。（三）建立根据地：如前所述，根据地就是我们已经了解的技术，成熟的流程和熟知的知识。根据地类似出发点，比如已经得到的一些非常可靠的结论，虽然不能完全解决问题，但已经被充分证明。经典物理的根据地是牛顿定律，但是要具体解决问题，还要扩大这个根据地，也就是发展进一步的定理定律。那些将要发展而还未完全确定的结论，就是所谓游击区。（四）战略防御和战略进攻：战略防御是保证已知的东西不再变成未知，斩落进攻是把未知转化成已知。（五）向运动战发展和（六）正确的指挥关系：见后面的讨论。

> 在游击战，速决性的要求是很大的，战役和战斗中包围敌人的外线圈则很小。......“集中大力，打敌小部”，仍然是游击战争战场作战的原则之一

为了达到速决，保卫敌人的外线圈需要很小，也就是说集中优势兵力限制敌人的活动范围和选项。具体到科研工作，这就需要严格控制变量。有的时候为了控制未知的变量不会干扰试验结果的解释，我们需要强迫症般的控制实验条件，一个比较极端但有时又很有必要的做法是copy exactly。这是英特尔公司为了保证生产线的稳定而采取的原则。游击战往往对应探索性科研，它的未知变量更多，如果没有特意的对变量的控制，实验结果就容易受到未知但重要的变量的随机变化而影响，给实验结果的解释造成困难。有时，我们并不知道某个条件是否重要，但是如果能去除它的潜在影响，也是缩小包围圈的方法。另外，特别是在探索性的科研工作中，对时效性要求较高，基于已有的知识，如果多个条件预期可以形成合力，那就可以考虑同时使用这些条线。

>游击战争要取得胜利，是不能离开它的计划性的。乱干一场的想法，只是玩弄游击战争，或者是游击战争的外行。不论是整个游击区的行动或是单个游击部队或游击兵团的行动，事先都应有尽可能的严密的计划，这就是一切行动的预先准备工作。情况的了解，任务的确定，兵力的部署，军事和政治教育的实施，给养的筹划，装备的整理，民众条件的配合等等，都要包括在领导者们的过细考虑、切实执行和检查执行程度的工作之中。没有这个条件，什么主动、灵活、进攻等事，都是不能实现的。固然正规战争的计划性更大些，游击战争的条件不容许很大的计划性，如果企图在游击战争中实行高度的严密的计划工作，那是错误的；但依照客观条件允许的程度，采取尽可能的严密的计划，则是必要的，须知同敌人斗争是一件不能开玩笑的事情

即使是探索性的工作，也要有尽可能严密的计划。把这里提到的条件放到科研工作中就是背景、目的和资源。这里还提到了领导的任务：过细考虑，切实执行和检查执行情况。所以，领导不能仅仅提出任务，还要有意愿有能力参与到具体的筹划当中来。这里也提到计划不是绝对的，而探索性工作的计划性更不能求全责备，但是一个尽可能严密的计划，才是后续的主动灵活的基础。

>这里虽说了许多的东西，但一切都环绕于战役和战斗的进攻。主动地位只有在进攻胜利之后，才能最后地取得。一切进攻战也都要主动地组织之，不要被迫地采取进攻。灵活地使用兵力，环绕于为着进攻战这个中心；计划性的必要，主要地也是为了进攻的胜利。战术上的防御手段，离开直接或间接协助进攻，则毫无意义。速决是说的进攻的时间，外线是说的进攻的范围。进攻是消灭敌人的唯一手段，也是保存自己的主要手段，单纯的防御和退却，对于保存自己只有暂时的部分的作用，对于消灭敌人则完全无用。

再次强调进攻的重要性。把这段话应用到科研工作中：一切的方针都是为了回答新问题发展新方法，要主动的面对新问题，而不是被动的等待问题出现。要灵活的创造有利于解决新问题的条件。制定计划也是为了成功的解决新问题。对已知问题的巩固，如果不是为了更好的解决新问题，则是没有意义的。要尽可能快的解决每个具体问题，而这些问题的范筹要通过控制变量来控制在一个可以速决的程度上。只有解决更多的问题，才能扩大自己的已知领域。

>游击战争的根据地是什么呢？它是游击战争赖以执行自己的战略任务，达到保存和发展自己、消灭和驱逐敌人之目的的战略基地。没有这种战略基地，一切战略任务的执行和战争目的的实现就失掉了依托。无后方作战，本来是敌后游击战争的特点，因为它是同国家的总后方脱离的。然而，没有根据地，游击战争是不能够长期地生存和发展的，这种根据地也就是游击战争的后方。......同流寇主义作思想斗争，将是一个不可少的过程。只有彻底地克服了流寇主义，提出并实行建立根据地的方针，才能有利于长期支持的游击战争。

探索性工作不能天马行空，要依托已知的技术和流程作为战略基地。根据地思想和流寇主义思想的斗争的问题，是任何游击战争中都会发生的，抗日游击战争在某种程度上也不能是例外。科研工作工作中的流寇主义往往更容易体现在名校毕业的高学历人才身上。这些人往往有很强的好奇心和很高的个人能力，但同时也可能不重视或者不屑于做那些看起来无聊的基础性工作。

>由此可知，从游击区到根据地，是一个艰难缔造的过程，依消灭敌人和发动民众的程度如何而定其是否已从游击区过渡到了根据地的阶段。

从探索性工作到成熟的技术需要经历很长的过程克服很多的困难。一次实验的成功并不能证明问题的解决，只有能够重复这个成功，并使之标准化、流程化才能算是真正掌握了它，使之成为根据地的一部分。


>正确的方针是巩固地向前发展，这是进可以攻退可以守的好办法。只要是长期战争，根据地的巩固和发展的问题，是每个游击队经常发生的问题。具体解决时应依照情况去决定。某一时期，把重心放在发展方面，这就是推广游击区、扩大游击队的工作。另一时期，则把重心放在巩固方面，这就是组织民众、训练部队的工作。

根据地是我们已经了解的技术，成熟的流程和熟知的知识。工作中有必要重复这些貌似已经“过时”的“无趣”的工作，因为这是巩固根据地所必须的。更重要的是，这里的巩固是以下一步的发展作为目的。比如巩固的流程可以体现在自动化和标准化，从而这个流程的费用会降低，成功率会更高，从而节省下资源以利于我们进行下一步发展，也就是向未知领域前进建立游击区。

>由是敌我各有加于对方的两种包围，大体上好似下围棋一样，敌对于我我对于敌之战役和战斗的作战好似吃子，敌之据点和我之游击根据地则好似做眼。

围棋这个例子很有意思，未知因素就像是“眼”。理论上讲，如果有两个眼，这块棋是杀不死的。放到实验中，就是说如果有两个变量在变化，实验结果是很难解释的。因此我们设计实验要要尽量控制变量，理想情况是只有一个变量在变化，那么实验结果就容易解释。

>部队的休息和训练是必要的，敌取守势时是我最好的休息和训练的时机。不是一事不做专门关起门来休息和训练，而是在扩大占领地，消灭小敌，发动民众的工作中，争取时间达到休息和训练的目的。解决给养被服等困难问题，也往往在这个时候。

所谓敌取守势，科研工作中的可能体现为不同具体情况。比如，有些实验本事需要耗时，一旦实验开始，除非仪器出现问题，结果虽然未知，但是基本上就确定了，也就是说“敌人”已经不能试驾影响了。再比如，项目的时间线相对充裕，紧迫性不高。这就给了我们巩固已有知识和技术的条件。这也体现在各个不同方面，比如重复已经成功的实验获得更多的统计数据和信心，进一步改进流程以提高效率降低成本。这些都会为将来的“进攻”提供一个好的基础。

>游击战争的指挥方法，由于游击部队是低级的武装组织和分散行动的特性，不容许高度的集中主义，如同正规战争的指挥方法那样。如果企图拿正规战争的指挥方法施之于游击战争，必然地要束缚游击战争的高度活泼性，而使游击战争毫无生气。高度的集中指挥和游击战争的高度活泼性是正相反对的东西；对于这种高度活泼的游击战争，施之以高度集中的指挥制度，不但不应该，而且不可能。

探索性的工作需要发挥主观能动性，因此不适合也不可能采取高度集中的指挥方式。也正因此，它也不适合大团队。

>游击战争的指挥原则，一方面反对绝对的集中主义，同时又反对绝对的分散主义，应该是战略的集中指挥和战役战斗的分散指挥。

战略层面上，探索性工作仍应服从整个团队的总体目标。具体的战役战斗层面上，应该允许更多的独立自主根据实际情况调整目标和手段。

>根据上述的原则，一般的方针集中于上级；具体的行动按照具体情况实施之，下级有独立自主之权。上级对下级某些具体行动有意见，可以而且应该作为“训令”提出，但决不应作为不可改变的“命令”​。越是地区广大，情况复杂，上下级距离很远，这种具体行动就越应加大其独立自主的权限，越应使之多带地方性，多切合地方情况的要求，以便培养下级和地方人员的独立工作能力，应付复杂的环境，发展胜利的游击战争。

这就很有操作性了。毕竟上级更多的了解宏观，和下级更多的了解微观。当探索性工作距离“根据地”很远，或者团队之间有技术上或者地理上的距离，下级的自主权就要更大，才能符合实际情况。同时，这也培养了下级工作人员的工作能力，给将才成长为帅才的机会。从这个角度讲，探索性工作给人的锻炼要更大，这也体现了研究性学习的有点。

>应该集中的不集中，在上者叫做失职，在下者叫做专擅，这是在任何上下级关系上特别是在军事关系上所不许可的。应该分散的不分散，在上者叫做包办，在下者叫做无自动性，这也是在任何上下级关系上特别是在游击战争的指挥关系上所不许可的。

本篇最后一句话振聋发聩。
