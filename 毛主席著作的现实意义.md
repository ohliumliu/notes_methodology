# 中国革命战争的战略问题
此文用翔实生动的事例给我们讲了深刻的哲学道理。辩证法不是毛主席发明的，但如果不能把它讲得如此具体生动，什么时候才能六亿神州尽尧舜？
我很早就看过马恩原著，两位导师都适用理论上的例子解释辩证法，比如微积分中体现的整体与局部的关系，量变与质变的关系等。
看了毛主席的文章，比如关于战略的持久性和战役的速决性的讨论，才领略到主席教学的魅力。

>对于敌人的一次有计划的“围剿”​，如果我们没有必要的和充分的准备，必然陷入被动地位。临时仓卒应战，胜利的把握是没有的。因此，在和敌人准备“围剿”同时，进行我们的反“围剿”的准备，实有完全的必要。我们队伍中曾经发生过的反对准备的意见是幼稚可笑的。
实际工作中，其实不一定是反对准备，而是没有足够水平去准备，以及也没有体会到准备的必要性。

# 抗日游击战争的战略问题（1938年5月）
虽然这篇文章讨论的是面对强大外敌侵略的战略问题，里面有写观点可以用在很多别的领域，比如科研工作中。虽然科技时刻在进步，但人类对大自然的理解还很有限。如果把大自然比作给我们出“难题”的对手，这个对手无疑是强大的，而我们自己的知识手段往往是有限的。“吾生也有涯，而知也无涯。以有涯随无涯，殆已！”教员的著作里，有很多值得学习的工具，让我们有机会摆脱“殆已”的宿命。

>为什么提起游击战争的战略问题

这是第一章的标题，解释了这篇文章的出发点。放到具体工作中，虽然大多数人都是牛马打工人，但毛主席多次强调，战略眼光和战术水平不是人人都有的，但是可以通过训练获得的。不管是谁，如果不使用正确的方法论，也是不可能得到正确的结论的。正确的方法论的来源就是对更高层次的战略问题的理解。有了对战略问题的理解，就有正确的发挥主观能动性的基础。另外，即使是从事比较专一具体的工作，也许在整个项目这个战略下，只算一个小的战役，但是这个战役本身还是要分解成更多的战斗。在战役和战斗的关系上，战役就又是战略层面的了。所以对战略问题的正确理解，也能帮助我们更有效的完成具体的工作。

>战争的基本原则是　保存自己消灭敌人

这是本篇第二章的标题，

>所以问题是：应该采取些什么方针或原则才能达到保存或发展自己和消灭敌人的目的呢？总的说来，主要的方针有下列各项：​（一）主动地、灵活地、有计划地执行防御战中的进攻战，持久战中的速决战和内线作战中的外线作战；​（二）和正规战争相配合；​（三）建立根据地；​（四）战略防御和战略进攻；​（五）向运动战发展；​（六）正确的指挥关系。这六项，是全部抗日游击战争的战略纲领，是达到保存和发展自己，消灭和驱逐敌人，配合正规战争，争取最后胜利的必要途径。

这些原则，很多都能在科研工作中找到对应。（一）防御战种的进攻战，持久战中的速决战和内线作战中的外线作战：人的知识往往有限而且彼此联系，就像是在一个圈子里，而圈子外面则是更加广袤的未知的需要探索的地方。因此，科研的出发点是保证已知的范围不再缩小或者已经获得的技能不会丢失，这就是防御战。但这个防御战里面必须有进攻的成分，不然就是重复劳动，不会增加我们的知识。又因为圈子外面的知识范围广深度大，这个探索过程必然是丑就的，即使是相对具体的产品研发，需要克服的困难也很多，也应该做打持久战的准备。当时持久战是由一个个速决战组成的，而这个速决战就是短期的任务，这个短期的任务要争取速战速决。如何才能速战速决的，一个原则就是集中优势兵力打歼灭战。在科研活动中，这个优势兵力往往体现在变量控制和操作复杂度。当实验只去研究一个或者很少的变量的效果时，结果就比较容易解释，从而更容易增加我们的知识。当实验操作比较容易时，也就不易犯错误，也更容易增加实验通量。这些条件，从战争角度讲，就是内线作战中的外线作战————虽然我们处在未知的包围下，属于内线作战，但在具体问题上，可以形成局部的反包围，实现相对多的已知面对相对少的位置，也就是局部的外线作战。（二）和正规战争相配合：游击战有点像科研里探索性的活动，它会帮助我们建立较小的分散的独立根据地，而正规战则是更加严格的瀑布式研发过程，遵循项目计划的预算和时间线。（三）建立根据地：如前所述，根据地就是我们已经了解的技术，成熟的流程和熟知的知识。根据地类似出发点，比如已经得到的一些非常可靠的结论，虽然不能完全解决问题，但已经被充分证明。经典物理的根据地是牛顿定律，但是要具体解决问题，还要扩大这个根据地，也就是发展进一步的定理定律。那些将要发展而还未完全确定的结论，就是所谓游击区。（四）战略防御和战略进攻：战略防御是保证已知的东西不再变成未知，斩落进攻是把未知转化成已知。（五）向运动战发展和（六）正确的指挥关系：见后面的讨论。

>正确的方针是巩固地向前发展，这是进可以攻退可以守的好办法。只要是长期战争，根据地的巩固和发展的问题，是每个游击队经常发生的问题。具体解决时应依照情况去决定。某一时期，把重心放在发展方面，这就是推广游击区、扩大游击队的工作。另一时期，则把重心放在巩固方面，这就是组织民众、训练部队的工作。

根据地是我们已经了解的技术，成熟的流程和熟知的知识。工作中有必要重复这些貌似已经“过时”的“无趣”的工作，因为这是巩固根据地所必须的。更重要的是，这里的巩固是以下一步的发展作为目的。比如巩固的流程可以体现在自动化和标准化，从而这个流程的费用会降低，成功率会更高，从而节省下资源以利于我们进行下一步发展，也就是向未知领域前进建立游击区。
